## 목적

## δ3 무게 예산 확정

> **상체 최대 6 kg까지 사용 가능. 배터리 위치(토르소/허리) 제약 없음.**

- δ2 적산값 ~4.2 kg → 완전히 안전 구간
- SO-ARM101 × 2 포함 설계 유지 가능
- 기획서 7.4절 기준: **"A1~A4 모두 안정 → 설계 여유 충분, 기능 추가 가능"** 해당

### δ2 상체 구성 요소별 무게

| 구성 요소 | 예상 무게 |
|---------|---------|
| 토르소 프레임 | 0.8 kg |
| 머리 (카메라 + 케이스) | 0.5 kg |
| SO-ARM101 × 2 | 1.6 kg |
| 배터리 (4S LiPo) | 0.8 kg |
| 전장 (보드, 케이블) | 0.5 kg |
| **합계** | **~4.2 kg** |

### 의사결정 기준 (기획서 7.4절)

| 결과 | 결정 |
|------|------|
| A1~A2에서만 안정 | 상부 4 kg 이하로 설계 제한 |
| A1~A3 안정 | 현재 설계(~4.2 kg) 유지 가능 |
| A1~A4 모두 안정 ← **실제 결과** | 설계 여유 충분, 기능 추가 가능 |
| A1도 불안정 | RL 재학습 or 하반신 설계 수정 필요 |
