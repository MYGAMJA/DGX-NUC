## 모니터링 판정 기준 (매번 확인)

```
✅ 정상
   - value loss: 유한값, 급등 없음
   - action std: > 0.1
   - feet_air_time: > 0 (0.001 이상이면 이상적)
   - NaN/Traceback: 0건

⚠️  경계 (주의 요망)
   - value loss: 급등/급락 반복
   - action std: 0.05~0.1 사이로 하락
   - feet_air_time: 0.0001 미만으로 하락

🛑 즉시 중단
   - value loss 또는 surrogate loss: nan
   - action std: 0.00 고정 (20 iter 이상)
   - feet_air_time: 0.0000 고정 (150 iter 이상)
   - Traceback 발생
```
