## 지금 이 정책으로 뭘 시킬 수 있나 (20번 내용 포함)

### 가능한 명령
- "X초 동안 앞으로 0.2 m/s로 걸어라"
- "X초 동안 왼쪽으로 비켜라"
- "X초 동안 제자리 회전해라"
- 위 조합: 앞으로 가면서 동시에 회전

예시 스케줄:
- 0~5초: 앞으로 0.2 m/s
- 5~8초: 앞으로 가면서 왼쪽으로 약간 이동
- 8~10초: 제자리에서 천천히 회전

### 아직 안 되는 것
- "저 좌표(2.3, -1.1)까지 가라" — **위치 명령은 현재 정책 밖의 일**
- 장애물 회피, 경로 계획
- → 이건 보행 엔진 위에 별도 경로 계획기를 붙여야 함

### 구조로 보면
```
(미래) 경로 계획기     → "저 위치까지 가라" 해석
(미래) 속도 변환기     → 위치 오차를 속도 명령으로 변환
(현재) 보행 정책       → "앞으로 0.2 m/s" 같은 속도 명령 실행
```
