# Hylion v6 외력 강건성 최종 목표: ±30N 달성 계획 (2026-04-17)

---

## 목표 선언

> **최종 목표: 외력 ±30N에서도 안정적으로 보행**
> - 현재 D 단계 최종(D5): ±10N
> - D5 완료 후 E 단계를 통해 ±30N 도달

---

## 왜 30N인가

| 기준 | 수치 | 비고 |
|------|------|------|
| 사람이 가볍게 미는 힘 | ~10N | 손가락으로 툭 |
| 사람이 한 손으로 미는 힘 | ~30N | 팔꿈치로 밀기 수준 |
| 데모 환경 외란 상한 | ~30N | 연구자가 시연 중 가할 수 있는 최대 충격 |
| 실제 로봇 전환 마진 | ±30N 학습 → 실제 ~15N까지 대응 | sim-to-real 보정 여유 포함 |

학습 시 30N을 견디게 훈련하면, 실제 로봇에서는 sim-to-real 손실이 있어도 15~20N 수준까지 안정적으로 대응 가능.

---

## 전체 로드맵

```
[완료] Stage A   → 기본 보행  (BHL 다리)
[완료] Stage B   → Hylion v6 전환 + 버그 수정
[완료] Stage B+  → 보상 39.18, orientation 3.27%
[완료] Stage C1  → +0.5kg 하중, orientation 3.84%
[진행] Stage D1  → 외력 ±1N   (3000 iter)  ← 현재
       Stage D2  → 외력 ±2N   (3000 iter)
       Stage D3  → 외력 ±3N   (4000 iter)
       Stage D4  → 외력 ±5N   (4000 iter)
       Stage D5  → 외력 ±10N  (5000 iter)  ← D 단계 완료 기준
         ↓
       Stage E1  → 외력 ±15N  (5000 iter)
       Stage E2  → 외력 ±20N  (6000 iter)
       Stage E3  → 외력 ±25N  (6000 iter)
       Stage E4  → 외력 ±30N  (7000 iter)  ← 최종 목표 ✨
```

---

## E 단계 상세 파라미터 설계

D 단계의 파라미터 패턴 (force : torque ≈ 3~3.5 : 1, 단계마다 mass 점진 증가)을 이어받아 설계.

| 스테이지 | 외력 | 토크 | mass 추가 | 반복 수 | 비고 |
|---------|------|------|----------|--------|------|
| D5 (현 목표) | ±10N | 3.0Nm | +1.5kg | 5000 | |
| **E1** | **±15N** | **5.0Nm** | **+1.5kg** | **5000** | 10→15 점프 |
| **E2** | **±20N** | **6.0Nm** | **+2.0kg** | **6000** | mass 증가 |
| **E3** | **±25N** | **7.5Nm** | **+2.0kg** | **6000** | |
| **E4** | **±30N** | **9.0Nm** | **+2.0kg** | **7000** | 최종 목표 |

**속도 범위 (E 단계 고정):**
- lin_vel_x = ±0.5 m/s (D 단계와 동일, 속도 확장은 별도 F 단계에서)
- standing_ratio = 0.10 (D5 수준 유지)

## E 단계 성공 기준

| 지표 | 기준값 | 근거 |
|------|-------|------|
| orientation 종료율 | < 15% | D 단계와 동일 |
| mean_reward | > 25.0 | 외력으로 인한 보상 감소 허용 |
| 에피소드 평균 길이 | > 300 스텝 | 최소 12초 이상 유지 |
